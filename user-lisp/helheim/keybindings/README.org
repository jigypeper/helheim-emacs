* Ediff

| Helheim     | Original   | Command                   |
|-------------+------------+---------------------------|
| =j=, =n=        | =n=, =SPC=     | ~ediff-next-difference~     |
| =k=, =N=        | =p=, =DEL=     | ~ediff-previous-difference~ |
| =d=           | =j=          | ~ediff-jump-to-difference~  |
| =gg=          | =1j=         | jump to first difference  |
| =G=           | =N/A=        | jump to last difference   |
| =l=, =a=        | =a=          | copy region A to region B |
| =h=, =b=        | =b=          | copy region B to region A |
| =C-e=         | =C-u 1 v=    | scroll down 1 line        |
| =C-y=         | =C-u 1 V=    | scroll up 1 line          |
| =C-d=, =v=, =C-v= | =v=, =C-v=     | scroll down half page     |
| =C-u=, =V=, =M-v= | =V=, =M-v=     | scroll up half page       |
| =zh=          | =>=          | scroll left               |
| =zl=          | =<=          | scroll right              |
| =H=           | =h=          | toggle highlighting       |
| =C-z=         | =z=          | ~ediff-suspend~             |
| =ua=, =ub=, =uc=  | =ra=, =rb=, =rc= | restore old diff          |
